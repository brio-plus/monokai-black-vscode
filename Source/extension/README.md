Monokai Black for Visual Studio Code
==================================

Monokai Black is a color scheme and icon set for Visual Studio Code. Focus on your code without eyes strain.

### Installation:

```
1. Extensions ‣ Search for 'Monokai Black' ‣ Install
2. Command Palette ‣ Preferences: Color Theme ‣ Monokai Black
2. Command Palette ‣ Preferences: File Icon Theme ‣ Monokai Black
```
